# FAQ

## ECMAScript 2026 {#ECMAScript2026}

::: details Чому обрана специфікація саме 2026?
Так як поточна версія специфікації може змінюватися, тому існує ненульова вірогідність того, що матеріали сайту не будуть відповідати поточній версії специфікації.
:::

## Console, console.log() {#console}

::: details
```console``` не є частиною специфікації ECMAScript (яка визначає сам JavaScript).
```console``` — це API середовища, а не частина мови. Тому середовища виконання JavaScript-коду (браузери, Node.js, Deno, Bun, і т.д.) реалізують його самостійно.

[WHATWG Console Standard](https://console.spec.whatwg.org/), [Node.js](https://nodejs.org/docs/latest/api/console.html), [Deno](https://docs.deno.com/api/node/console/), [Bun](https://bun.com/reference/globals/Console)
::: 



## DevTools {#DevTools}

::: details Як відкрити DevTools в браузері?

Комбінації клавіш для відкриття DevTools:

- Chrome DevTools: F12 або Ctrl+Shift+I (Win) / Cmd+Option+I (Mac)

- Firefox Developer Tools / Web Developer Tools: F12 або Ctrl+Shift+I

- Web Inspector (Safari): Cmd+Option+I (потрібно спочатку увімкнути в налаштуваннях)

- Microsoft Edge DevTools: F12 або Ctrl+Shift+I

- Developer Tools (Opera): дивись Chrome DevTools

Наразі мобільні браузери не мають DevTools!
:::
