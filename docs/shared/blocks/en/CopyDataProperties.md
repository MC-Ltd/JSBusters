      <h1><span class="secnum">7.3.25</span> CopyDataProperties ( <var>target</var>, <var>source</var>, <var>excludedItems</var> )</h1>
      <p>The abstract operation CopyDataProperties takes arguments <var>target</var> (an Object), <var>source</var> (an <emu-xref href="#sec-ecmascript-language-types" id="_ref_2645"><a href="ecmascript-data-types-and-values.html#sec-ecmascript-language-types">ECMAScript language value</a></emu-xref>), and <var>excludedItems</var> (a <emu-xref href="#sec-list-and-record-specification-type" id="_ref_2646"><a href="ecmascript-data-types-and-values.html#sec-list-and-record-specification-type">List</a></emu-xref> of <emu-xref href="#property-key" id="_ref_2647"><a href="ecmascript-data-types-and-values.html#property-key">property keys</a></emu-xref>) and returns either a <emu-xref href="#sec-completion-record-specification-type" id="_ref_2648"><a href="ecmascript-data-types-and-values.html#sec-completion-record-specification-type">normal completion containing</a></emu-xref> <emu-const>unused</emu-const> or a <emu-xref href="#sec-completion-record-specification-type" id="_ref_2649"><a href="ecmascript-data-types-and-values.html#sec-completion-record-specification-type">throw completion</a></emu-xref>. It performs the following steps when called:</p>
      <emu-alg><ol><li><span aria-hidden="true" class="list-marker">1. </span>If <var>source</var> is either <emu-val>undefined</emu-val> or <emu-val>null</emu-val>, return <emu-const>unused</emu-const>.</li><li><span aria-hidden="true" class="list-marker">2. </span>Let <var>from</var> be !&nbsp;<emu-xref aoid="ToObject" id="_ref_2650"><a href="abstract-operations.html#sec-toobject">ToObject</a></emu-xref>(<var>source</var>).</li><li><span aria-hidden="true" class="list-marker">3. </span>Let <var>keys</var> be ?&nbsp;<span class="e-user-code"><var>from</var>.<var class="field">[[OwnPropertyKeys]]</var>()</span>.</li><li><span aria-hidden="true" class="list-marker">4. </span>For each element <var>nextKey</var> of <var>keys</var>, do<ol><li><span aria-hidden="true" class="list-marker">   a. </span>Let <var>excluded</var> be <emu-val>false</emu-val>.</li><li><span aria-hidden="true" class="list-marker">   b. </span>For each element <var>e</var> of <var>excludedItems</var>, do<ol><li><span aria-hidden="true" class="list-marker">      i. </span>If <emu-xref aoid="SameValue" id="_ref_2651"><a href="abstract-operations.html#sec-samevalue">SameValue</a></emu-xref>(<var>e</var>, <var>nextKey</var>) is <emu-val>true</emu-val>, then<ol><li><span aria-hidden="true" class="list-marker">         1. </span>Set <var>excluded</var> to <emu-val>true</emu-val>.</li></ol></li></ol></li><li><span aria-hidden="true" class="list-marker">   c. </span>If <var>excluded</var> is <emu-val>false</emu-val>, then<ol><li><span aria-hidden="true" class="list-marker">      i. </span>Let <var>desc</var> be ?&nbsp;<span class="e-user-code"><var>from</var>.<var class="field">[[GetOwnProperty]]</var>(<var>nextKey</var>)</span>.</li><li><span aria-hidden="true" class="list-marker">      ii. </span>If <var>desc</var> is not <emu-val>undefined</emu-val> and <var>desc</var>.<var class="field">[[Enumerable]]</var> is <emu-val>true</emu-val>, then<ol><li><span aria-hidden="true" class="list-marker">          1. </span>Let <var>propValue</var> be ?&nbsp;<emu-xref aoid="Get" id="_ref_2652"><a href="abstract-operations.html#sec-get-o-p" class="e-user-code">Get</a></emu-xref>(<var>from</var>, <var>nextKey</var>).</li><li><span aria-hidden="true" class="list-marker">          2. </span>Perform !&nbsp;<emu-xref aoid="CreateDataPropertyOrThrow" id="_ref_2653"><a href="abstract-operations.html#sec-createdatapropertyorthrow">CreateDataPropertyOrThrow</a></emu-xref>(<var>target</var>, <var>nextKey</var>, <var>propValue</var>).</li></ol></li></ol></li></ol></li><li><span aria-hidden="true" class="list-marker">5. </span>Return <emu-const>unused</emu-const>.</li></ol></emu-alg>
      <emu-note><span class="note">Note</span><div class="note-contents">
        <p>The target passed in here is always a newly created object which is not directly accessible in case of an error being thrown.</p>
      </div></emu-note>
