      <p>Абстрактна операція CopyDataProperties приймає аргументи <var>target</var> (Object ❮Об'єкт❯), <var>source</var> (an <emu-xref href="#sec-ecmascript-language-types" id="_ref_2645"><a href="ecmascript-data-types-and-values.html#sec-ecmascript-language-types">ECMAScript language value ❮значення мови ECMAScript❯</a></emu-xref>), та <var>excludedItems</var> (<emu-xref href="#sec-list-and-record-specification-type" id="_ref_2646"><a href="ecmascript-data-types-and-values.html#sec-list-and-record-specification-type">List ❮Список❯</a></emu-xref> з <emu-xref href="#property-key" id="_ref_2647"><a href="ecmascript-data-types-and-values.html#property-key">property keys ❮ключів властивостей❯</a></emu-xref>) і повертає або <emu-xref href="#sec-completion-record-specification-type" id="_ref_2648"><a href="ecmascript-data-types-and-values.html#sec-completion-record-specification-type">normal completion containing ❮ >>>нормальне завершення, що містить <<< ❯</a></emu-xref> <emu-const>unused</emu-const> або <emu-xref href="#sec-completion-record-specification-type" id="_ref_2649"><a href="ecmascript-data-types-and-values.html#sec-completion-record-specification-type">throw completion ❮ >>> завершення з помилкою <<< ❯</a></emu-xref>.
