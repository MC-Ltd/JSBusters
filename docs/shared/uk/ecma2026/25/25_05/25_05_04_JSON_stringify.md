        <pre><code class="javascript hljs">a = [];

        <p>Рядкові значення обгортаються <u>>>> code unit/коде-юнітом/кодові одиниці <<<</u> QUOTATION MARK ❮ЛАПКИ❯ (<code>"</code>). <u>>>> Code units/коде-юніти/коде-юнітси/кодові одиниці <<<</u> <code>"</code> та <code>\</code> екрануються префіксом <code>\</code>. <u>>>> Code units/коде-юніти/коде-ютінтси/кодові одиниці <<<</u> керуючих символів замінюються керуючими послідовностями <code>\u</code>HHHH, або коротшими формами, <code>\b</code> (BACKSPACE ❮ЗВОРОТНІЙ ПРОБІЛ❯), <code>\f</code> (FORM FEED ❮СИМВОЛ НОВОЇ СТОРІНКИ❯), <code>\n</code> (LINE FEED ❮СИМВОЛ НОВОГО РЯДКА❯), <code>\r</code> (CARRIAGE RETURN ❮ПОВЕРНЕННЯ КАРЕТКИ❯), <code>\t</code> (CHARACTER TABULATION ❮СИМВОЛ ТАБУЛЯЦІЇ❯).</p>
