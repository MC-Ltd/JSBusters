      <h1><span class="secnum">7.3.16</span> TestIntegrityLevel ( <var>O</var>, <var>level</var> )</h1>
      <p>The abstract operation TestIntegrityLevel takes arguments <var>O</var> (an Object) and <var>level</var> (<emu-const>sealed</emu-const> or <emu-const>frozen</emu-const>) and returns either a <emu-xref href="#sec-completion-record-specification-type" id="_ref_2557"><a href="ecmascript-data-types-and-values.html#sec-completion-record-specification-type">normal completion containing</a></emu-xref> a Boolean or a <emu-xref href="#sec-completion-record-specification-type" id="_ref_2558"><a href="ecmascript-data-types-and-values.html#sec-completion-record-specification-type">throw completion</a></emu-xref>. It is used to determine if the set of own properties of an object are fixed. It performs the following steps when called:</p>
