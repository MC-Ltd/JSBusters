        <h1><span class="secnum">25.5.4.6</span> SerializeJSONArray ( <var>state</var>, <var>value</var> )</h1>
        <p>The abstract operation SerializeJSONArray takes arguments <var>state</var> (a <emu-xref href="#sec-json-serialization-record" id="_ref_15626"><a href="structured-data.html#sec-json-serialization-record">JSON Serialization Record</a></emu-xref>) and <var>value</var> (an <emu-xref href="#sec-ecmascript-language-types" id="_ref_15627"><a href="ecmascript-data-types-and-values.html#sec-ecmascript-language-types">ECMAScript language value</a></emu-xref>) and returns either a <emu-xref href="#sec-completion-record-specification-type" id="_ref_15628"><a href="ecmascript-data-types-and-values.html#sec-completion-record-specification-type">normal completion containing</a></emu-xref> a String or a <emu-xref href="#sec-completion-record-specification-type" id="_ref_15629"><a href="ecmascript-data-types-and-values.html#sec-completion-record-specification-type">throw completion</a></emu-xref>. It serializes an array. It performs the following steps when called:</p>
        <emu-alg><ol><li><span aria-hidden="true" class="list-marker">1. </span>If <var>state</var>.<var class="field">[[Stack]]</var> contains <var>value</var>, throw a <emu-val>TypeError</emu-val> exception because the structure is cyclical.</li><li><span aria-hidden="true" class="list-marker">2. </span>Append <var>value</var> to <var>state</var>.<var class="field">[[Stack]]</var>.</li><li><span aria-hidden="true" class="list-marker">3. </span>Let <var>stepBack</var> be <var>state</var>.<var class="field">[[Indent]]</var>.</li><li><span aria-hidden="true" class="list-marker">4. </span>Set <var>state</var>.<var class="field">[[Indent]]</var> to the <emu-xref href="#string-concatenation" id="_ref_15630"><a href="ecmascript-data-types-and-values.html#string-concatenation">string-concatenation</a></emu-xref> of <var>state</var>.<var class="field">[[Indent]]</var> and <var>state</var>.<var class="field">[[Gap]]</var>.</li><li><span aria-hidden="true" class="list-marker">5. </span>Let <var>partial</var> be a new empty <emu-xref href="#sec-list-and-record-specification-type" id="_ref_15631"><a href="ecmascript-data-types-and-values.html#sec-list-and-record-specification-type">List</a></emu-xref>.</li><li><span aria-hidden="true" class="list-marker">6. </span>Let <var>len</var> be ?&nbsp;<emu-xref aoid="LengthOfArrayLike" id="_ref_15632"><a href="abstract-operations.html#sec-lengthofarraylike" class="e-user-code">LengthOfArrayLike</a></emu-xref>(<var>value</var>).</li><li><span aria-hidden="true" class="list-marker">7. </span>Let <var>index</var> be 0.</li><li><span aria-hidden="true" class="list-marker">8. </span>Repeat, while <var>index</var> &lt; <var>len</var>,<ol><li><span aria-hidden="true" class="list-marker">   a. </span>Let <var>strP</var> be ?&nbsp;<emu-xref aoid="SerializeJSONProperty" id="_ref_15633"><a href="structured-data.html#sec-serializejsonproperty" class="e-user-code">SerializeJSONProperty</a></emu-xref>(<var>state</var>, !&nbsp;<emu-xref aoid="ToString" id="_ref_15634"><a href="abstract-operations.html#sec-tostring">ToString</a></emu-xref>(<emu-xref aoid="𝔽" id="_ref_15635"><a href="notational-conventions.html#%F0%9D%94%BD">𝔽</a></emu-xref>(<var>index</var>)), <var>value</var>).</li><li><span aria-hidden="true" class="list-marker">   b. </span>If <var>strP</var> is <emu-val>undefined</emu-val>, then<ol><li><span aria-hidden="true" class="list-marker">      i. </span>Append <emu-val>"null"</emu-val> to <var>partial</var>.</li></ol></li><li><span aria-hidden="true" class="list-marker">   c. </span>Else,<ol><li><span aria-hidden="true" class="list-marker">      i. </span>Append <var>strP</var> to <var>partial</var>.</li></ol></li><li><span aria-hidden="true" class="list-marker">   d. </span>Set <var>index</var> to <var>index</var> + 1.</li></ol></li><li><span aria-hidden="true" class="list-marker">9. </span>If <var>partial</var> is empty, then<ol><li><span aria-hidden="true" class="list-marker">   a. </span>Let <var>final</var> be <emu-val>"[]"</emu-val>.</li></ol></li><li><span aria-hidden="true" class="list-marker">10. </span>Else,<ol><li><span aria-hidden="true" class="list-marker">    a. </span>If <var>state</var>.<var class="field">[[Gap]]</var> is the empty String, then<ol><li><span aria-hidden="true" class="list-marker">       i. </span>Let <var>properties</var> be the String value formed by concatenating all the element Strings of <var>partial</var> with each adjacent pair of Strings separated with the code unit 0x002C (COMMA). A comma is not inserted either before the first String or after the last String.</li><li><span aria-hidden="true" class="list-marker">       ii. </span>Let <var>final</var> be the <emu-xref href="#string-concatenation" id="_ref_15636"><a href="ecmascript-data-types-and-values.html#string-concatenation">string-concatenation</a></emu-xref> of <emu-val>"["</emu-val>, <var>properties</var>, and <emu-val>"]"</emu-val>.</li></ol></li><li><span aria-hidden="true" class="list-marker">    b. </span>Else,<ol><li><span aria-hidden="true" class="list-marker">       i. </span>Let <var>separator</var> be the <emu-xref href="#string-concatenation" id="_ref_15637"><a href="ecmascript-data-types-and-values.html#string-concatenation">string-concatenation</a></emu-xref> of the code unit 0x002C (COMMA), the code unit 0x000A (LINE FEED), and <var>state</var>.<var class="field">[[Indent]]</var>.</li><li><span aria-hidden="true" class="list-marker">       ii. </span>Let <var>properties</var> be the String value formed by concatenating all the element Strings of <var>partial</var> with each adjacent pair of Strings separated with <var>separator</var>. The <var>separator</var> String is not inserted either before the first String or after the last String.</li><li><span aria-hidden="true" class="list-marker">       iii. </span>Let <var>final</var> be the <emu-xref href="#string-concatenation" id="_ref_15638"><a href="ecmascript-data-types-and-values.html#string-concatenation">string-concatenation</a></emu-xref> of <emu-val>"["</emu-val>, the code unit 0x000A (LINE FEED), <var>state</var>.<var class="field">[[Indent]]</var>, <var>properties</var>, the code unit 0x000A (LINE FEED), <var>stepBack</var>, and <emu-val>"]"</emu-val>.</li></ol></li></ol></li><li><span aria-hidden="true" class="list-marker">11. </span>Remove the last element of <var>state</var>.<var class="field">[[Stack]]</var>.</li><li><span aria-hidden="true" class="list-marker">12. </span>Set <var>state</var>.<var class="field">[[Indent]]</var> to <var>stepBack</var>.</li><li><span aria-hidden="true" class="list-marker">13. </span>Return <var>final</var>.</li></ol></emu-alg>
        <emu-note><span class="note">Note</span><div class="note-contents">
          <p>The representation of arrays includes only the elements in the <emu-xref href="#interval" id="_ref_15639"><a href="notational-conventions.html#interval">interval</a></emu-xref> from <emu-val>+0</emu-val><sub>𝔽</sub> (inclusive) to <code>array.length</code> (exclusive). Properties whose keys are not <emu-xref href="#array-index" id="_ref_15640"><a href="ecmascript-data-types-and-values.html#array-index">array indices</a></emu-xref> are excluded from the stringification. An array is stringified as an opening LEFT SQUARE BRACKET, elements separated by COMMA, and a closing RIGHT SQUARE BRACKET.</p>
